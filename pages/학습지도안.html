  • 실생활 예시 (스팸 메일, 집 가격 예측) 소개

💻 실습 체험 - AI 모델 학습 및 예측

실습 1: 분류 모델 (15분)

  • 주제: 시험 합격/불합격 예측
  • 데이터: 공부시간, 수면시간 → 시험결과
  • 활동: AI 모델 학습 버튼 클릭, 정확도 확인
  • 체험: 새로운 데이터로 예측 해보기

실습 2: 회귀 모델 (5분)

  • 주제: 부동산 가격 예측
  • 데이터: 평수, 건물연수 → 가격
  • 활동: 3D 시각화로 데이터 분포 확인

교사 활동

📊 지도학습 vs 비지도학습

구분 지도학습 비지도학습
데이터 정답 있음 정답 없음
목적 예측 패턴 발견
예시 스팸 분류, 가격 예측 고객 세분화, 추천 시스템
15분

💻 실습 체험 - 클러스터링 실습

고객 세분화 실습

  • 데이터: 고객의 나이, 연소득, 온라인구매점수
  • 목표: 비슷한 특성의 고객 그룹 찾기
  • 활동 순서:
    1. 고객 데이터 분포 확인 (5분)
    2. 클러스터 수 선택 및 그룹 찾기 (5분)
    3. 발견된 그룹별 특성 분석 (5분)

🎯 기대 학습 성과

  • 데이터에서 숨겨진 패턴을 발견하는 과정 체험
  • 그룹별 특성을 해석하고 마케팅 전략 이해
  • 실무에서 AI가 활용되는 방식 이해
5분

📝 형성평가 - 학습 내용 확인

평가 구성

  • 객관식 3문제: 지도/비지도학습 개념 (3분)
  • 성찰 작성: 학습 소감 자유 서술 (2분)
  • 제한 시간: 총 5분
  • 자동 채점: 즉시 결과 확인

📊 평가 기준

  • 우수 (80점 이상): 개념을 정확히 이해하고 적용
  • 양호 (60-79점): 기본 개념 이해
  • 보통 (60점 미만): 추가 학습 필요
5분

🎊 마무리 - 전체 학습 정리

교사 활동

  • 교사 대시보드로 전체 학습 현황 확인
  • 우수 학습자 격려 및 피드백
  • AI 기술의 미래와 진로 연계 안내

학생 활동

  • 전체 학습 완료 확인
  • 동료와 학습 경험 공유
  • AI 분야 진로 관심도 표현

🌟 학습 마무리 포인트

  • AI는 데이터로부터 학습한다
  • 문제에 따라 적절한 학습 방법 선택
  • AI 기술은 우리 생활 곳곳에서 활용
  • 지속적인 관심과 학습이 중요

📊 평가 계획

📝 형성평가 (즉시 평가)

평가 방법

  • 온라인 객관식 퀴즈 (3문제)
  • 학습 성찰 서술형 (1문제)
  • 자동 채점 및 즉시 피드백

평가 기준

등급 점수 기준
우수 80-100점 개념 정확히 이해, 적용 가능
양호 60-79점 기본 개념 이해
보통 60점 미만 추가 학습 필요

👥 수행평가 (과정 중심)

평가 영역

참여도 (30%)
  • 플랫폼 적극적 활용
  • 실습 활동 참여
  • 질문 및 토론 참여
이해도 (40%)
  • 개념 설명 능력
  • 실습 결과 해석
  • 사례 적용 능력
협력성 (30%)
  • 동료와 협력 학습
  • 지식 공유 의지
  • 상호 도움 제공
